O que o JavaScript é capaz de fazer?

O JavaScript pode ser usado para as seguintes coisas:

  • Parte front-end de sites com animações e intereções;
  • Para fazer jogos;
  • Parte back-end de sites com o Node.Js;
  • Para criar aplicativos mobile com o React Native.

Algumas Informações Sobre A Linguagem:

  • Quando foi criada: 1995;
  • Quem criou: Brendan Eich;
  • Tipagem: Dinâmica;
  • Nível: Alto nível;
  • Paradigmas: POO / Funcinal / Procedural / Orientada a eventos.
